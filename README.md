# Скрипт для генерации паролей

```bash
usage: generate-password [-h] [--length LENGTH] [--special]

Password generator

options:
  -h, --help            show this help message and exit
  --length LENGTH, -l LENGTH
                        Password length
  --special, -s         Include special characters
```

### Установка

1) для Linux:
```bash
chmod +x install.sh
./install.sh
```
